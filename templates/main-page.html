Countries API

This API has been created just to help students to practice AJAX transactions

Current endpoints for this API:

  • /countries

    provides a list of countries found in the database.

  • /country/{name}/

    provides the country description (lots of info here)

    Example: /country/brazil

  • /countries/language/{language}

    provides a list of countries that speaks a given language

    Example: /countries/language/portuguese

  • /countries/region/{region}

    provides a list of countries found in a given region

    Example: /countries/region/africa

    Possible regions: Europe, Asia, Africa, Oceania, Americas, Antarctic
